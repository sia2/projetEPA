Installation de la dernière mise à jour

Il est impératif d'ajouter de nouveau champs dans la table "connexion" pour que miniBB fonctionne.

Pour cela, vous trouverez le fichier "alter_table_connexion.sql" qui vous permettra d'ajouter les nouveaux champs à la table "connexion".

Il vous suffit simplement d'importer ce fichier dans la base de données "bddepa".

Une fois l'importation terminée, vous pouvez tester en ajoutant plusieurs utilisateurs dans la table "connexion". 

Vous pourrez désormais vous connecter via la page développée par le groupe 4 et accéder accéder au forum et vice versa.

Bon courage.
